# 🧑‍💻 Сайт-визитка / Портфолио

Добро пожаловать в мой персональный сайт-портфолио! Это одностраничный сайт, созданный, чтобы представить мои навыки, проекты и увлечения в сфере программирования и преподавания.

**Живая версия:** [https://caxapok91.github.io/site2/](https://caxapok91.github.io/site2/)

## 📖 О проекте

Этот сайт — моя цифровая визитная карточка. Он создан, чтобы в удобной и наглядной форме рассказать о моём опыте как разработчика и преподавателя в школе программирования **Itmouse**.

Цель проекта — объединить в одном месте информацию обо мне, моём стеке технологий и ключевых работах, а также продемонстрировать мои навыки вёрстки и создания интерактивных интерфейсов.

## ✨ Основные функции

*   **Адаптивный дизайн:** Сайт корректно отображается на любых устройствах: от смартфонов до настольных мониторов.
*   **Тёмная тема:** Встроенный переключатель позволяет выбрать светлый или тёмный режим оформления, который запоминается в браузере.
*   **Интерактивные карточки проектов:** Каждый проект можно "развернуть", нажав на карточку, чтобы увидеть подробное описание.
*   **Интеграция с внешними сервисами:** Ссылка на песню, созданную с помощью Suno AI, демонстрирует творческий подход.

## 🛠️ Технологии

Сайт построен с использованием современных веб-технологий:

*   **HTML5:** Семантическая структура страницы.
*   **CSS3:** Стилизация с использованием CSS-переменных, Flexbox, Grid и анимаций. Реализована поддержка тёмной темы без использования JavaScript-фреймворков.
*   **JavaScript (нативный):** Обеспечивает интерактивность: раскрытие карточек проектов, переключение тем и сохранение выбранной темы в `localStorage`.
*   **Font Awesome:** Иконки для визуального оформления.
*   **Google Fonts:** Шрифт Inter для современного и читаемого текста.
